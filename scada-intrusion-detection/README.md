## HMM Based IDS for SCADA
```
#Dependencies
 - python
 - hmmlearn (0.2.1)
 - numpy (>=1.11)
 - Cython 
 - pandas
 - statsmodels
 - scipy
 - matplotlib

Contains src code for HMM Based IDS system for SCADA networks. It 
considers system data level features and network level features
and models them as multivariate time series for anomaly detection.
```
